HTML

  • Stands for HyperText Markup Language
  • A computer language that defines the structure of raw text displayed on a computer or a device
  • Syntax
    • <opening tag>content</closing tag>
  • Elements
    • Head Element: Contatins metadata to assit the browser to correctly render the page
    • Body Element: Defines the body and content for the user to see

CSS

  • Stands for Cascading Style Sheets
  • Define style rules for specific elements of an HTML page and format them using a wide range of style options
  • CSS rules contain a selector and declaration block, which includes properties and value
  • The CSS Box Model includes content, padding, border, and margin

Git

  • An open-source version control software; GitHub is the web-based interface
  • Git Commands
    • git status: confirms the current branch
    • git add -A: adds the changed files in current working directory to the staging area
    • git commit -m //message: commit to repo
    • git pull origin main: ensures the current branch is up to date before opening a pull request
    • git push origin //curent branch with changes: pushes up all of our changes

JavaScript

  • A programming language that allows interaction with websites
  • Case-sensitive
  • Variable: Named container that stores data
  • Array: Single variable used to hold a group of data
  • Control Flow: the order in which the computer executes code in a script
